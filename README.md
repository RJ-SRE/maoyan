# 猫眼电影 - 简易版

模仿猫眼电影网界面,用 Vue 3 实现简易版（https://www.maoyan.com）。
实现该网站的前端页面跳转、数据读取、数据传递等效果。

## 项目结构

```
maoyan/
├── src/              # 主应用文件
├── assets/           # 静态资源
├── components/       # 公共组件
├── router/           # 路由配置
├── store/            # 状态管理
├── view/             # 页面视图
└── view/sub-view/    # 子页面
```

## 功能特性

### 1. 顶部导航栏
- 猫眼电影图标和品牌名称
- 导航菜单: 主页、电影、影院、演出
- APP下载链接
- 登录按钮

### 2. 主页
- 正在热映电影展示
- 即将上映电影展示
- 电影卡片包含海报、评分、类型、上映时间
- 购票和预售功能

### 3. 电影页面
- 三个导航标签: 正在上演、即将上映、经典影片
- 详细电影信息展示(导演、演员、评分、简介)
- 购票/想看按钮

### 4. 影院页面
- 城市选择器
- 影院列表展示
- 每个影院包含: 名称、地址、最低票价、距离
- 影院特色标签(IMAX、杜比、4D等)
- 选座购票功能

### 5. 演出页面
- 城市选择器
- 演出分类: 音乐会、话剧歌剧、舞蹈芭蕾、体育赛事
- 演出详情: 海报、场馆、时间、票价
- 购票功能

### 6. 底部信息栏
- 关于我们、用户帮助、合作伙伴、关注我们等链接
- 版权信息

## 技术栈

- Vue 3
- Vue Router 4
- Vuex 4
- Vite

## 安装与运行

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 预览生产构建
```bash
npm run preview
```

## 开发说明

本项目为学习和演示目的创建,仅供参考使用。使用了占位图片作为示例数据。

实际应用中需要:
1. 连接真实的API获取电影、影院、演出数据
2. 实现用户认证和登录功能
3. 实现实际的购票流程
4. 添加更多交互功能和页面
